# 🩸 Blood Management System

A web-based Blood Management System developed using **Python Flask**, **MongoDB**, **HTML**, **CSS**, **Bootstrap**, and **JavaScript**. This system helps connect blood donors with recipients and streamlines the blood donation process.

## 🚀 Features

* 👤 User Registration and Login
* 🩸 Blood Donor Registration
* 🏥 Blood Request Management
* 📩 Share Donor Details with Recipients
* 🔐 JWT Authentication
* 📧 Email Verification using OTP
* 📊 Admin Dashboard with Statistics and Charts
* ✅ Request Status Management
* 📱 Responsive User Interface using Bootstrap

## 🛠️ Technologies Used

### Backend

* Python
* Flask
* MongoDB
* PyMongo
* JWT Authentication

### Frontend

* HTML
* CSS
* Bootstrap
* JavaScript
* Chart.js

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/noumi553/Blood-Management-System.git
cd Blood-Management-System
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux/Mac

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

The application will start at:

```text
http://localhost:8000
```
